# FSD论文实验进度记录

## 1. IEEE-CIS实验结果

### 1.1 数据集信息
- 节点: 100,000
- 边: 4,765,352
- 特征: 394
- 欺诈率: 3.58%
- 平均度: 47.65

### 1.2 GNN方法性能对比

| 方法 | AUC | F1 | Seeds |
|------|-----|-----|-------|
| **H2GCN** | **0.8180±0.0027** | 0.184 | 3 |
| **GraphSAGE** | **0.8164±0.0032** | 0.189 | 3 |
| **MixHop** | **0.8127±0.0051** | 0.178 | 3 |
| NAA-GCN | 0.7488±0.0024 | 0.169 | 3 |
| GCN | 0.7463±0.0046 | 0.168 | 10 |
| GAT | 0.7441 | 0.157 | 1 |

**关键发现**: GraphSAGE/H2GCN/MixHop (~0.81) 显著优于 GCN/GAT/NAA-GCN (~0.74)，差距约7%

### 1.3 图诊断指标

| 指标 | 值 | 含义 |
|------|-----|------|
| ρ_FS^(1) | 0.0584 | 1-hop特征-结构对齐 |
| ρ_FS^(2) | 0.0277 | 2-hop更不相似 |
| Homophily h | 0.9306 | 高同配性 |
| δ_agg (度1-10) | 0.74 | 低聚合稀释 |
| δ_agg (度200+) | 33.59 | 高聚合稀释(45倍) |

### 1.4 FSD预测分析

- **原预测**: ρ_FS=0.059 → "无明显赢家"
- **实际结果**: 采样/拼接方法显著更优
- **原因**: FSD未考虑度分布导致的聚合稀释

## 2. 论文改进方向

### 2.1 原定位
- NAA是更好的方法
- ρ_FS预测最优GNN方法

### 2.2 新定位 (更强贡献)
- **多维度GNN诊断框架**
- 从单指标ρ_FS扩展到多维度诊断

### 2.3 新理论贡献

1. **聚合稀释度 δ_agg**
   - 定义: E[d_i · (1 - sim(x_i, mean(x_neighbors)))]
   - 高δ_agg → 采样/拼接方法更优

2. **多维度决策框架**
   ```
   ρ_FS高? → NAA方法
        ↓No
   Homophily低? → H2GCN
        ↓No
   度方差高? → GraphSAGE/拼接方法
        ↓No
   标准GCN/GAT
   ```

3. **IEEE-CIS深度案例研究**
   - 高同配性 + 高度数 = 需要特殊处理
   - 揭示了单指标诊断的局限性
